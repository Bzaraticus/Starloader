First launch settings

Default paths:
Windows: C:\Program Files (x86)\Steam\SteamApps\common\Starbound
OS X: ~/Library/Application Support/Steam/SteamApps/common/Starbound
Linux: ~/.local/share/Steam/SteamApps/common/Starbound

A good choice is <starbound's path>/mods, e.g.
~/.local/share/Steam/SteamApps/common/Starbound/mods

If this folder doesn't exist, we will ask you if we can create it automatically.
